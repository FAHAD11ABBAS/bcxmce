## ðŸš€ How to Run this Project Locally

### Prerequisites
- [LocalWP](https://localwp.com/) installed on your machine.
- [Git](https://git-scm.com/) installed.

### Installation Steps
1. **Clone the repository:**
   ```bash
   git clone https://github.com/FAHAD11ABBAS/bcxmce.git
   ```
2. **Open LocalWP** and create a NEW site (e.g., named "bcxmce").
3. **Go to the site folder:**
   Right-click the new site in LocalWP -> "Reveal in Finder/Explorer".
4. **Replace files:**
   Go to `app/public` in the new site folder and delete its contents.
   Copy the contents of `app/public` from this repository into that folder.
5. **Database (Optional/Advanced):**
   If a database dump (`.sql`) is provided in `app/sql`, you can import it using the "Open Adminer" / "Database Manager" button in LocalWP.

### Auto-Open Feature
This project includes scripts to automatically open the browser when the site starts:
- **Mac/Linux**: Run `./start_project.sh`
- **Windows**: Run `start_project.bat`
